Project Overview

Designed and built an automated coin dispenser with a 3D-printed enclosure and a motorized pushing mechanism. The project integrated mechanical design, electronics, and embedded programming to create a functional device that demonstrates precision control and automation.

Key Features

  • Automated dispensing of coins via a servo-driven pusher mechanism.
  • Custom 3D-printed container for durability and precise component alignment.
  • Integration of ultrasonic sensors to detect user input for dispensing coins.
  • Reliable and repeatable operation for consistent coin output.

Technical Implementation

The coin dispenser was built around a servo motor connected to a gear-driven pusher that releases coins from a container. An ultrasonic sensor detects when a hand is present, triggering the motor via an Arduino microcontroller programmed in C++. The enclosure was designed in CAD and 3D-printed, ensuring precise alignment of components. Connections were established with breadboard wiring to maintain flexibility during testing. Iterative testing and adjustment optimized the timing and motor movement to ensure smooth dispensing without jamming.

Technologies Used:

  • Hardware: Arduino microcontroller, servo motor, ultrasonic sensor, custom 3D-printed parts.
  • Software: Arduino IDE (C++) for programming control logic.
  • Design: Fusion360 CAD, Tinkercad
  • Fabrication: 3D printing for container, breadboard wiring for prototyping.

Challenges & Solutions

  • Coin jamming: Initial mechanism sometimes stuck; solved by redesigning the pusher geometry and adjusting alignment.
  • Timing precision: Motor movement needed fine-tuning; resolved by calibrating the Arduino code for accurate dispense cycles.
  • Stability and durability: Breadboard connections initially shifted during testing; ensured reliable operation by carefully securing components and wiring.

Results

The final prototype successfully dispensed coins on demand, demonstrating smooth, reliable operation and effective integration of mechanical, electronic, and programming components. The project showcased practical skills in automation, CAD-based design, and embedded system control.
