Autonomous Google Dino Game Player

← Back to Portfolio

Project Overview

Built an autonomous system that plays the Google Chrome Dino Game by using a light sensor to detect obstacles and a motorized actuator to press the spacebar. The project combined computer vision, mechanical control, and automation, showcasing how hardware can interact with software environments.

Key Features

  • Automated gameplay using a sensor to detect on-screen obstacles.
  • Motor-driven actuator to press the spacebar and control the Dino’s jump.
  • Real-time obstacle detection for responsive inputs.
  • Integrated hardware-software interaction between sensor data and mechanical motion.

Technical Implementation

The system used a light sensor to scan the game screen for incoming obstacles. When something was detected, an Arduino microcontroller programmed in C++ triggered a motorized actuator that physically pressed the keyboard’s spacebar. Timing logic was implemented to ensure the jump occurred at the correct moment, with repeated calibration cycles to improve detection accuracy and response speed. The hardware was mounted securely to ensure consistent actuation without mispressing keys.

Technologies Used:

  • Hardware: Arduino microcontroller, light sensor, DC motor/servo with actuator arm.
  • Software: Arduino IDE (C++) for sensor input processing and motor control.
  • Design: Fusion360 CAD, Tinkercad
  • Processes: Real-time detection, mechanical actuation design, calibration testing.

Challenges & Solutions

  • Detection accuracy: Initial false positives/negatives from the sensor; solved by refining the detection threshold and calibrating sensor sensitivity.
  • Timing precision: Motor movement needed fine-tuning; resolved by calibrating the Arduino code for accurate dispense cycles.
  • Mechanical reliability: Early prototypes misaligned with the spacebar; resolved by redesigning the actuator mount for stability.

Results

The final system successfully played the Dino Game autonomously, consistently detecting obstacles and jumping at the correct time. The project highlighted practical skills in automation, hardware-software integration, and creative mechanical design.
